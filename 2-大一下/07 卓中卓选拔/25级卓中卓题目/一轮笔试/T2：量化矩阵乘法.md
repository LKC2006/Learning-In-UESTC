# T2：量化矩阵乘法

在深度学习推理中，模型量化技术将浮点权重和激活值转换为低比特整数（如 int8），从而大幅减少模型体积、提升计算速度。实现高质量的量化算子需要深入理解量化原理、数值范围和定点计算。

请你实现一个 int8 量化矩阵乘法，支持量化模式：per-tensor（整个张量共享量化参数）。你需要处理量化/反量化过程，确保计算结果与浮点矩阵乘的误差在允许范围内。

请实现函数：

```c
// 模式 ：per-tensor 量化矩阵乘法
// A: MxK 浮点矩阵，B: KxN 浮点矩阵，C: MxN 输出矩阵（已分配）
void quant_matmul_tensor(const float *A, const float *B, float *C, int M, int N, int K);
```

要求：

- 提交普通矩阵乘法属欺骗行为，一经发现，取消选拔资格。

- 量化方案：采用对称量化（zero point = 0），int8 数值范围 [-128, 127]。

- 量化参数计算： Per-tensor：计算整个矩阵的绝对值最大值 max_abs，量化系数 scale = max_abs / 127.0。

- 量化过程：浮点值 x 量化：q = round(x / scale)，并饱和到 [-128, 127]。

- 整数矩阵乘法：对每个输出元素 (i,j)，计算累加和： sum_q = sum (qA[i][k] * qB[k][j])其中 `qA` 和 `qB` 为量化后的 int8 矩阵。累加和 sum_q 为 int32 类型。

  k=0

  K-1

- 反量化： C[i][j] = scale_a * scale_b * sum_q。

- 精度验证：你的实现应使结果与浮点矩阵乘（直接使用浮点乘法计算）的误差小于 0.01。

- 禁止事项：不允许在整数乘法阶段使用浮点运算，所有乘法必须在量化整数域内完成。

- 内存管理：可以动态分配临时矩阵，但需确保无内存泄漏。

输入输出

```
输入矩阵 A、B 按行主序存储，均为 float 类型。
矩阵维度：1 ≤ M,N,K ≤ 512，数值范围：-50.0 ≤ 元素 ≤ 50.0。
输出矩阵 C 已分配，大小为 M * N，直接写入结果。
```

程序运行框架如下：

```c
// ------------------------------- 测试主函数 -------------------------------------------
int main() {
    int M, N, K;
    // 输入格式：第一行为 M N K
    if (scanf("%d %d %d", &M, &N, &K) != 3) {
        fprintf(stderr, "Error reading dimensions and mode.\n");
        return 1;
    }
    // 分配内存并读取矩阵 A (M x K)
    float *A = (float*)malloc(M * K * sizeof(float));
    if (!A) { perror("malloc A"); return 1; }
    for (int i = 0; i < M * K; ++i) {
        if (scanf("%f", &A[i]) != 1) {
            fprintf(stderr, "Error reading A matrix.\n");
            free(A);
            return 1;
        }
    }
    // 读取矩阵 B (K x N)
    float *B = (float*)malloc(K * N * sizeof(float));
    if (!B) { perror("malloc B"); free(A); return 1; }
    for (int i = 0; i < K * N; ++i) {
        if (scanf("%f", &B[i]) != 1) {
            fprintf(stderr, "Error reading B matrix.\n");
            free(A); free(B);
            return 1;
        }
    }
    // 分配输出矩阵 C (M x N)
    float *C = (float*)malloc(M * N * sizeof(float));
    if (!C) { perror("malloc C"); free(A); free(B); return 1; }
    quant_matmul_tensor(A, B, C, M, N, K);
    // 输出结果矩阵 C (每行 N 个浮点数，空格分隔，行末换行)
    for (int i = 0; i < M; ++i) {
        for (int j = 0; j < N; ++j) {
            printf("%d", (int)(C[i * N + j]*100));
            if (j < N - 1) printf(" ");
        }
        printf(" ");
    }
    // 释放内存
    free(A);
    free(B);
    free(C);
    return 0;
}
```
