# **T1：位图 + 位压缩头部内存分配器**

操作系统的内存管理模块需要追踪哪些内存页已被分配、哪些仍然空闲。业界常用两种紧凑结构实现这一目标：

- **位图（Bitmap）**：用一个二进制位表示一页的分配状态，极度节省空间
- **压缩块头部**：将"块大小""分配状态""合法性校验"等多个字段打包进一个 `uint32_t`，通过位运算存取

本题要求在一个固定大小的内存池上，综合运用**位运算**、**指针算术**、**首次适应算法**，实现一个完整的页式内存分配器。

内存池整体布局如下：

```
内存池共 10240 字节，划分为 320 页，每页 32 字节：

页号：  0        1        2        3     ...    319
      ┌────────┬────────┬────────┬────────┬────────┐
      │[头4字节 │        │        │        │        │
      │+数据区] │  数据   │  数据   │  数据  │  数据   │
      └────────┴────────┴────────┴────────┴────────┘
         ↑
       每个块的第一页存放4字节块头部（BlockHdr）
       后续各页均为纯数据区

块头部（BlockHdr）= uint32_t，共32位，字段分布如下：

 bit:  31      30        16  15        8   7        0
       ┌───────┬───────────┬───────────┬────────────┐
       │is_free│  pages    │ (保留)    │   MAGIC    │
       │ 1 bit │  15 bits  │   8 bits  │   8 bits   │
       └───────┴───────────┴───────────┴────────────┘
          │         │                        │
          │         │                        └── 固定值 0xA5
          │         │                            用于检测头部合法性
          │         └────────────────────────── 本块连续占用的页数
          └──────────────────────────────────── 1=空闲，0=已分配

位图（page_bitmap）：uint32_t 数组，共10个元素（320位对应320页）

bit:  319  318  317  ...   1   0
     ┌────┬────┬────┬───┬───┬───┐
     │P319│P318│P317│...│ P1│ P0│
     └────┴────┴────┴───┴───┴───┘
        每一位对应一页：1 = 已分配，0 = 空闲
```

相关定义如下：

```c
#include <stdio.h>
#include <stdint.h>
#include <string.h>

/* ══ 参数定义 ══════════════════════════════════════ */
#define PAGE_SIZE    32
#define PAGE_COUNT   320
#define POOL_SIZE    (PAGE_SIZE * PAGE_COUNT)
#define BITMAP_WORDS ((PAGE_COUNT + 31) >> 5)

/* ══ 块头部位字段常量 ══════════════════════════════ */
typedef uint32_t BlockHdr;

#define HDR_MAGIC        0xA5u
#define HDR_MAGIC_MASK   0x000000FFu// bits [7:0]
#define HDR_FREE_MASK    0x80000000u// bit  [31]
#define HDR_FREE_SHIFT   31
#define HDR_PAGES_MASK   0x7FFF0000u// bits [30:16]
#define HDR_PAGES_SHIFT  16

/* ══ 全局变量 ════════════════════════════════════ */
uint8_t  memory_pool[POOL_SIZE];
uint32_t page_bitmap[BITMAP_WORDS];

/* ══ 地址与页号互转宏 ═════════════════════════════ */
#define PAGE_ADDR(pg)   (memory_pool + (pg) * PAGE_SIZE)
#define ADDR_PAGE(ptr)  (((uint8_t*)(ptr) - memory_pool) / PAGE_SIZE)
#define HDR_OF(pg)      ((BlockHdr*)PAGE_ADDR(pg))

void *mem_alloc(int size);
void mem_free(void *ptr);

/* ══ 辅助函数（已实现，可直接调用） ══════════════════ */

// 检查头部 MAGIC 是否合法
int hdr_check_magic(BlockHdr h) {
    return (h & HDR_MAGIC_MASK) == HDR_MAGIC;
}

// 打印位图
void bitmap_dump(void) {
    unsigned long *p = (unsigned long *)page_bitmap;
    printf("%lu\n", *p);
}

// 初始化内存池
void mem_init(void) {
    memset(memory_pool, 0, sizeof(memory_pool));
    memset(page_bitmap, 0, sizeof(page_bitmap));
    // 整个内存池作为一个空闲块，写入头部
    *HDR_OF(0) = HDR_FREE_MASK
                 | ((uint32_t)PAGE_COUNT << HDR_PAGES_SHIFT)
                 | HDR_MAGIC;
}
```
